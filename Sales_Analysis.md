# Sales Performance Dashboard

## Overview
This project is an interactive Sales Performance Dashboard created using Microsoft Excel. The dashboard helps analyze sales data through charts, pivot tables, KPI cards, and slicers.
## Dataset used
- <a href="https://github.com/AkulaNavyaSri08/Sales_Dashboard/blob/main/Sales%20Analysis%20Dashboard.xlsx">Dataset view</a>
## Features
- Total Revenue Analysis
- Total Units Sold
- Revenue by Product
- Revenue by Region
- Sales Trend Analysis
- Interactive Slicers for Filtering
- Pivot Tables & Pivot Charts
- KPI Cards for Quick Insights

## Tools & Technologies Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- KPI Cards
- Data Visualization Techniques

## Dashboard Components
### KPI Cards
- Total Revenue
- Total Units Sold

### Charts
- Revenue by Product (Column Chart)
- Revenue by Region (Pie Chart)
- Revenue Trend (Line Chart)
- Salesperson Performance (Bar Chart)

### Filters
- Region
- Product
- Salesperson

## Dataset Information
The dataset contains:
- Date
- Region
- Product
- Units Sold
- Unit Price
- Revenue
- Salesperson
